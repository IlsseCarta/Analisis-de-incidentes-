### ⚠️ Descripción   

La empresa ficticia **Yummy Recipes For Me** sufrió un ataque de denegación de servicio (DoS) provocado por una avalancha de paquetes ICMP, originada por una configuración incorrecta del firewall. Este incidente interrumpió los servicios de red durante aproximadamente dos horas.

En el presente reporte se describen las estrategias implementadas para contener, mitigar y prevenir este tipo de incidentes, incluyendo análisis de tráfico de red, ajustes en las reglas de firewall y mejoras en los controles de seguridad.


English version

### ⚠️ Incident Description  

A simulated organization, **Yummy Recipes For Me**, experienced a Denial of Service (DoS) attack caused by an ICMP flood due to a firewall misconfiguration, resulting in a two-hour network outage.

This report outlines the analysis, mitigation strategies, and security improvements implemented to restore service availability and prevent similar incidents.
